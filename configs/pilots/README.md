# Pilot Configurations

This directory contains runnable or historical pilot configurations. The project root keeps only
`config.yaml` as the deterministic default.

- `deepseek.smoke.yaml`: small DeepSeek-compatible smoke configuration.
- `practice_*.yaml`: practice dataset pilots.
- `qianfan_product_boundary_*.yaml`: product-boundary experiments.
- `qianfan_rag_v2_focused.yaml`: focused RAG evaluation.
- `qianfan_a5_multiturn_*.yaml`: A5 multi-turn intake pilots.
- `qianfan_vendors_smoke.yaml`: cross-vendor smoke configuration.

The planned 450-output experiment remains in `../experiments/` and is not a completed result.
